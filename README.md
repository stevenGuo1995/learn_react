# learn_react
学习react的使用

### 配置cnpm

#### 安装cnpm

```bash
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

#### 配置npm 淘宝源

```bash
npm config set registry https://registry.npm.taobao.org
```

### 新建 ts 格式的 react app

```bash
npx --ignore-existing create-react-app . --template typescript
```
